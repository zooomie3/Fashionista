# Fashionista

A code that would acts as a "decider" on what to wear on specific occasions (classes). 

The following classes are:
- weather
- occasion
- what is in your closet

The code will take the data and by analyzing, will ouput an outfit proposal. We are planning to make this proposal a visualization (by using images of the clothes). It would should an avatar wearing such clothes. Another visualization idea, would be making a bar plot, showing the correlation between warmth and clothes.
